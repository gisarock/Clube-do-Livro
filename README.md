Seu Clube do Livro Online!
Descubra seu próximo livro favorito com apenas um clique!
O que faz essa aplicação?

Já pensou em ter uma biblioteca particular sempre à mão? Com o nosso Clube do Livro, você pode pesquisar por seus livros favoritos de forma rápida e fácil! Basta digitar o título ou algumas palavras-chave e pronto: uma lista com diversas opções incríveis aparecerá para você!

Como funciona a mágica?

Por trás dessa experiência incrível, temos uma combinação de tecnologias que trabalham juntas em harmonia:

HTML: A estrutura básica da página, como você vê os títulos, parágrafos e botões.
CSS: O estilista da nossa página, responsável por deixar tudo bonitinho e organizado.
JavaScript: O cérebro da operação! É ele quem faz a busca, compara as informações e mostra os resultados na tela.
Como usar?

Digite: Escreva o nome do livro ou algumas palavras-chave no campo de busca.
Clique: Aperte o botão "Pesquisar" e prepare-se para se surpreender com os resultados!
Explore: Clique nos links para saber mais sobre cada livro e onde comprá-lo.
Para desenvolvedores:

Se você é um curioso da programação, sinta-se à vontade para explorar o código! A gente usou um array chamado "dados" para armazenar todas as informações dos livros. Cada item desse array tem um objeto com as seguintes propriedades:

livro: O título do livro.
resumo: Uma breve descrição do livro.
link: Um link para a Amazon (ou outra loja online) onde você pode comprar o livro.
tags: Palavras-chave para ajudar na busca (por exemplo, "autoajuda", "ficção científica", "programação").
